      Computer software

      Description

      cURL is a computer software project providing a library and command-line tool for transferring data using various protocols. It was first released in 1997. The name stands for "Client URL". The original author and lead developer is the Swedish developer Daniel Stenberg. Wikipedia
      License: Free Software: MIT/X derivate license
      Original author(s): Daniel Stenberg
      Written in: C
      Stable release: 7.65.1 (5 June 2019; 2 months ago)
      Platforms: x86-32 (32 bit Intel x86), x86-64
